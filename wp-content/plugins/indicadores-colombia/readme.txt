== Installation ==
Esta sección explica como instalar y activar el plugin.

Ejemplo:

1. cargue `igniweb-indicadoresCO.php` en la carpeta `/wp-content/plugins/indicadores-colombia/`

2. Active el plugin a traves de la opción 'Plugins' del menu en WordPress

3. En la opción 'Widgets' desplace 'Indicadores Económicos - CO' hacia el Area donde lo desea visualizar.
